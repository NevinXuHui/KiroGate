🚀 最快启动方式：

  🪟 Windows 用户：
     ✅ 双击 "启动.bat" 文件（最简单！）
     ✅ 或运行 "run.bat"
     ✅ 或运行 ".\run.ps1"（PowerShell）

  🐧 Linux 用户：
     ✅ ./run.sh

  🍎 macOS 用户：
     ✅ ./run.sh

📋 常用命令：

  ./run.sh              启动服务
  ./run.sh dev          开发模式（热重载）
  ./run.sh docker       Docker 部署
  ./run.sh install      安装依赖
  ./run.sh check        检查环境
  ./run.sh help         查看帮助

  （Windows 用户将 ./run.sh 替换为 run.bat）
